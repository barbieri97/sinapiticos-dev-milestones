📖 Como este material foi construído?

Este recurso é baseado no artigo científico «Evidence-Informed Milestones for Developmental Surveillance Tools» (Zubler et al., 2022, Pediatrics), publicado pela Academia Americana de Pediatria. Os marcos foram revisados por 8 especialistas e passaram por critérios rigorosos antes de serem incluídos.

📊

Critério dos 75%

Só entrou na lista o marco que pelo menos 75% das crianças já alcançaram naquela idade. Isso evita a postura de "esperar para ver", pois não é a média — é uma referência mais exigente.

🔬

Base em evidências

Cerca de 80% dos marcos têm suporte de dados normativos de pelo menos uma fonte científica. Os outros têm apoio de ferramentas de triagem validadas ou opinião clínica publicada.

⚠️

Fim dos "sinais de alerta" separados

A nova lista eliminou a coluna separada de alertas. Se a criança não atingiu um marco que 75% das crianças já alcançaram, isso por si só já é um sinal para investigar mais.

🧠

4 domínios do desenvolvimento

Os marcos são organizados em: Social-Emocional, Linguagem e Comunicação, Cognitivo e Motor — para facilitar a vigilância em diferentes áreas.

⚠️ Importante: Estes checklists não substituem triagens validadas como o ASQ-3 ou o M-CHAT-R. Eles são uma ferramenta de vigilância — para orientar conversas entre famílias e profissionais e apoiar o julgamento clínico sobre quando fazer triagens adicionais.

🔎 Por que não usar a mediana (50%)? Se usássemos a idade média, metade das crianças ainda não teria alcançado o marco — o que levaria o profissional a dizer "é normal" quando talvez houvesse um risco. O critério de 75% é mais sensível sem ser excessivamente restritivo.
